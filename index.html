ElectricalPy

ElectricalPy is a python module made for Electrical engineers inspired by PSPICE. The module is capable of solving any electrical circuit with any number of passive elements and any number of dependent or independent sources. The module can also plot the circle diagram of induction motor from the data of Open circuit test & Short circuit test. Besides these, it can also perform various small but time consuming tasks on the go. Click here to view the project.

Library Automation

The project is divided into three sections:

  1. Live tracking of available books
  2. Issuing of books
  3. Providing security

Live tracking is acheived by a robot that goes around the book shelves and scans books that are already equipped with passive RFID tags and then updates an excel file which shows the records of books available at present and there respective positions.

Similarly issuing of books is carried out by swiping RFID based ID card along with books that are to be issued, along with these tasks the system also mantains the record of issue and return dates and calculates fine amount if there is any deadline violation.
